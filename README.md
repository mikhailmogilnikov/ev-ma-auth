# Auth Form Example

## Requirements
- `bun` >= 1.2.0

## How to start
- Install dependencies `bun i`
- Start dev server `bun run dev`
- For production version execute `bun run build` and `bun run preview`

## Tech stack
- `TypeScript`
- `React v19`
- `Vite`
- `Tailwind CSS`
- `React Hook Form`
- `Zod`
- `MSW` (API Mocking)
